Meu corpo 我嘅身體 我的身體 My body

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini, Ingrid Schechter

Priscilla Freitas de Oliveira

Alfredo Ferreira

葡萄牙語

Level 1

Posso correr com ele.

我可以同佢跑。

讓我能跑步。

I can run with it.

Posso pular com ele.

我可以同佢跳。

讓我能跳躍。

I can jump with it.

Posso dançar com ele.

我可以同佢跳舞。

讓我能跳舞。

I can dance with it.

Posso nadar com ele.

我可以同佢游水。

讓我能游泳。

I can swim with it.

Posso saltar com ele.

我可以同佢跳繩。

讓我能跳繩。

I can skip with it.

Posso chutar com ele.

我可以同佢踢波。

讓我能踢球。

I can kick with it.

Posso sair com ele.

我可以同佢逃走。

讓我能逃走。

I can escape with it.

Mas nunca poderei voar com ele.

但係永遠都唔可以同佢飛！

可是總不能讓我飛！

But I can never fly with it.

作者: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
插圖: Mlungisi Dlamini, Ingrid Schechter
譯文: Priscilla Freitas de Oliveira
講述: Alfredo Ferreira
