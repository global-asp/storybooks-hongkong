My body 我嘅身體 我的身體

Clare Verbeek, Thembani Dladla, Zanele Buthelezi

Mlungisi Dlamini

Darshan Soni

English

Level 1

I can run with it.

我可以同佢跑。

讓我能跑步。

I can run with it.

I can jump with it.

我可以同佢跳。

讓我能跳躍。

I can jump with it.

I can dance with it.

我可以同佢跳舞。

讓我能跳舞。

I can dance with it.

I can swim with it.

我可以同佢游水。

讓我能游泳。

I can swim with it.

I can skip with it.

我可以同佢跳繩。

讓我能跳繩。

I can skip with it.

I can kick with it.

我可以同佢踢波。

讓我能踢球。

I can kick with it.

I can escape with it.

我可以同佢逃走。

讓我能逃走。

I can escape with it.

But I can never fly with it.

但係永遠都唔可以同佢飛！

可是總不能讓我飛！

But I can never fly with it.

Written by: Clare Verbeek, Thembani Dladla, Zanele Buthelezi
Illustrated by: Mlungisi Dlamini
Read by: Darshan Soni
Language: English
Level 1
